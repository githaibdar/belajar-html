Membahas tentang tag html

Di dalam tag head ada beberapa tag yang bisa disimpan:

Judul halaman menggunakan tag title

CSS menggunakan tag style

javascript menggunakan tag script

metadata menggunakan tag meta

metadata dalam tag head cukup penting untuk memberikan informasi web, diantaranya

charset untuk encoding

decription, keywords dan author yang berguna untuk memudahkan search engine dalam mengidentifikasi website yang kita buat

Tag yang bisa dimasukkan dalam tag body diantaranya:

Teks, pendukung teks, gambar, hyperlink, list, table, form, script, object dan grouping

Tag penting lainnya adalah tag komentar

struktur dari tag adalah nama tag, atribut dan value/nilai
