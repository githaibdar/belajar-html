Atribut type utuk tag ol memiliki beberapa value/nilai yang digunakan untuk mengganti jenis penomoran menggunakan angka arab, huruf atau angka romawi

  1. vaule 1 merupakan value default, penomoran menggunakan angka arab
  2. value "A" merupakan penomoran menggunakan huruf kapital
  3. value "a" merupakan penomoran menggunakan huruf kecil
  4. value "I" merupakan penomoran romawi kapital
  5. value "i" merupakan penomoran romawi kecil

Berikut ini adalah tag ul yang berarti unordered list/list tidak berurut

  • Buku
  • Majalah
  • Piringan

ul juga memiliki atribut type dengan value sebagai berikut:

  • value "disc"
  • value "square"
  • value "circle"

pada contoh di atas, value yang digunakan pada ul dan ol disimpan pada masing masing tag li. Untuk memudahkan saat menggunakan satu value type, simpan type pada tag ul atau ol, sehingga berlaku pada semua tag li
