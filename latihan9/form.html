FORM

Tag untuk membuat form adalah "form". Didalam form masih ada tag lagi yang menjadi elemen-elemen form, diantaranya adalah:

  • input

    • Digunakan untuk meng-input sesuatu

  • textarea

    • Untuk memasukkan text yang panjang, misalnya komentar, email, posting Facebook

  • select

    • Digunakan untuk membuat dropdown

  • button

    • Tombol untuk mengirimkan data atau melakukan hal spesifik

  • label

    • Untuk menandai sebuah elemen input

Input

Tag input disertai atribut type yang digunakan bagi kita untuk memilih membuat input tipe apa. Type input yaitu:

  • Text
  • Password
  • Radio
  • Checkbox
  • Submit
  • Reset
  • Button

Input text

Atribut text digunakan untuk meng-input teks. Misalnya kita akan membuat form username. Pada pembuatan form, jangan gunakan tag "p" untuk tulisan tapi gunakan tag "label". Didalam tag label gunakan atribut "for" untuk menandai label tersebut digunakan untuk form apa. Misalnya kita akan gunakan "username" untuk value "for" nya. Setelah itu buat "text" field-nya dengan tag "input" dan attribut "text". Tambahkan juga atribut "id" dan isi valuenya sama dengan value atribut for pada tag label, yaitu "username".

Atribut for dan id ini digunakan untuk menghubungkan tag label dan text. Fungsinya adalah saat meng-klik tulisan yang dibuat dengan tag label, kita bisa langsung mengisi text field input text-nya. Contohnya seperti ini.

Hal yang sama berlaku juga untuk password. Perbedaannya, kata yang dimasukkan pada kolom password tidak akan ditampilkan. Contohnya seperti ini.

Selanjutnya adalah radio type atribut radio digunakan untuk membuat form pilihan, misalnya pria/wanita, bekerja/pelajar dan lain sebagainya. Cara membuatnya sama seperti tag input lainnya. Misalnya kita ingin membuat form pilihan jenis kelamin pria dan wanita. Kita buat dulu tag input untuk pilhan prianya menggunakan atribut type dengan value radio, gunakan juga atribut "id" untuk terhubung dengan tag label nanti. Pada input radio ini perlu ditambahkan atribut name. Atribut name ini berguna untuk mengelompokkan beberapa tag input radio agar pilihan yang dipilih nanti hanya bisa satu saja. Value name yang digunakan harus sama dengan tag lainnya yang satu kelompok. Dalam cotoh ini kita akan menggunakan name dengan value "jenis-kelamin" agar nanti hanya bisa memilih hanya satu diantara pria dan wanita. Setelah membuat tag pria berisi atribut type, id dan name baru buat label-nya agar posisi label ada di sebelah kanan pilihan. Kemudian buat tag input dan label untuk pilihan wanita, dan jangan lupa gunakan value "jenis-kelamin" untuk atribut name. Hasilnya akan menjadi seperti ini, hanya salah satu saja yang bisa dipilih

Seanjutnya form pilihan untuk memilih lebih dari satu pilihan menggunakan "checkbox". Caranya sama seperti "radio", Buat dulu "input"-nya baru kemudian labelnya agar kotak check berada di depan keterangan pilihan.

Setelah membuat isian form selanjutnya adalah membuat tombol kirim/submit, reset atau tombol lainnya. Untuk membuat itu dapat menggunakan input dengan attribut "submit, reset dan button". Misalnya kita akan membuat tombol submit, untuk membuat tulisan tombolnya menjadi kirim maka gunakan juga attribut value yang isinya "kirim". Hasilnya seperti ini.

textarea
